# SunTheme11

Windows 11 için gün doğumu ve gün batımına göre Açık/Koyu temayı otomatik değiştiren hafif yardımcı uygulama.

## Özellikler
- Gün doğumu/gün batımına göre otomatik tema
- IP tabanlı yaklaşık konum
- Çevrimdışı son konum desteği
- Hibrit tema algılama
- Otomatik başlangıç
- Sessiz arka plan çalışması
- Windows 11 x64 ve ARM64
- MIT Lisansı

## Kurulum
1. Uygun sürümü indirin.
2. ZIP'i çıkarın.
3. Install.bat çalıştırın.

## Gizlilik
Telemetri, reklam ve kullanıcı takibi yoktur.
