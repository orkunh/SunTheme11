## Neden SunTheme11?

Windows yalnızca belirlediğiniz saatlerde tema değiştirebilir.

SunTheme11 bulunduğunuz konumdaki gerçek gün doğumu ve gün batımını takip eder.

Böylece saat ayarlamanıza gerek kalmaz.
