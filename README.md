# SunTheme11

Automatic Windows 11 Light/Dark theme switching based on local sunrise and sunset.

## Features
- Automatic sunrise/sunset theme switching
- Automatic IP-based approximate location
- Offline cached location
- Hybrid theme detection
- Automatic startup
- Silent background operation
- Windows 11 x64 & ARM64
- MIT License

## Installation
1. Download the correct release.
2. Extract the ZIP.
3. Run Install.bat.

## Privacy
No telemetry, advertising or user tracking.
