## Features

✅ Automatic Light/Dark switching

✅ Local sunrise & sunset calculation

✅ Automatic approximate location detection

✅ Offline cached location

✅ Hybrid theme detection

✅ Silent background operation

✅ Windows 11 x64

✅ Windows 11 ARM64

---

## Why SunTheme11?

Windows can automatically switch themes on a schedule.

SunTheme11 automatically follows your local sunrise and sunset instead.

No manual schedule required.
